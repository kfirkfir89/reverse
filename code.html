                        //Reverse the string
                        //Logic function
                        function reverseString(userString){
                        
                            let revString = [];
                            //reverse a string using a for loop
                            for (let index = userString.length - 1; index >= 0; index--) {
                                revString += userString[index];
                            }
                            return revString;
                        
                        }
                        
                        //Display the reverse string to the user
                        //View Function
                        function displayString(revString){
                        
                            //wrtie to the page
                            document.getElementById("msg").innerHTML = `Your string reversed is: ${revString}`
                            //show the alert box
                            document.getElementById("alert").classList.remove("invisible");
                        
                        }

The Code is structed in one function

getValues

getValues gets the user input from the page. it utilizes getElementById to pull the values from the page. it stores it in userString and then passes those values to the reverseString function.
The function reverseString returns an array of values and passes that array to the displayString function to display the revString array.

reverseString

reverseString takes in userString array. We create new array revString to stores the new reversed array.
A for loop is used to pull the values to the new array. The loop start counting from the last index of userString array and pull the strings in reverse to the new revString array.

displayString

displayString takes in revString array and display the reversed array in alert box that shows up when display.
The alret is using the classList.remove to show up the alert window message by remove the class "invisible".
